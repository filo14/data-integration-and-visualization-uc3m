# Database

- Start docker database container

    ```bash
    make up
    ```

- Stop docker database container

    ```bash
    make down
    ```

- Apply database schema

    ```bash
    make schema
    ```

- Create datbase backup

    ```bash
    make backup
    ```

- Enter database shell

    ```bash
    make connect
    ```

- Clean backup folder

    ```bash
    make clean
    ```
